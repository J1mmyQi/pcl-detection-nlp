This directory stores historical experiment material that is no longer needed in
the main working tree.

Contents:

- `configs/`: archived exploratory configuration files (`exp_*.json`), older
  RoBERTa trial configs, and the offline RoBERTa fallback configuration.
- `artifacts/`: archived outputs for exploratory runs, legacy baselines, and
  one-off search summaries that were useful for tuning but are not part of the
  final submission surface.

The main repository root keeps only the selected configurations and the
currently relevant artifacts needed for the final report and submission.
