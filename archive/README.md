This directory stores historical project material that is no longer needed on
the main working surface.

Contents:

- `basic_tests.ps1`: the older PowerShell smoke-test helper, kept only as a
  secondary convenience script now that the repository is notebook-first.
- `configs/`: archived exploratory configuration files (`exp_*.json`) and
  older RoBERTa trial configs that are no longer part of the active config set.
- `artifacts/`: archived outputs for exploratory runs and legacy baselines that
  are no longer part of the final submission surface.

The main repository root keeps only the selected configurations and the
currently relevant artifacts needed for the final report and submission.
